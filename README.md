https://samrawit-tsehaye.github.io/WebDesign_PersonalPortfolio/

## About This Project

This is a responsive personal portfolio website designed to present:

* My background and goals
* Certificates I’ve earned
* Practice projects I’ve built
* My contact and social links

## 🛠️ Built With

HTML5
CSS3
Font Awesome
Google Fonts (Poppins)

##  Features

* Clean and modern design
* Header with background image
* Right-aligned navigation bar
* Centered welcome section
* Certificates displayed in a grid
* Hover effects on images
* Project showcase with links
* Social media contact section
